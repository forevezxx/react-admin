# react-admin
react-admin system solution

##### 1.启动初始化
```js
// 首推荐使用yarn装包
yarn or npm i
```
##### 2.启动项目
```js
yarn start or npm start
```
##### 3.打包项目
```js
yarn build or npm run build
npm run build
```
##### 4.项目发布
